How-To Guide for Google Maps Info Window API

Functions - More Info Link

Below is a list of the functions that can be used to create, delete, or edit the link to more information in the location's info window.

Function Description
location.setInfoLink(url: string, [text: string], [order: number], [new-window: boolean]) Return: None
This function sets the more info link field of the location's info window. It is not required for the user to set the more info link. If no link is chosen, then the field will not appear in the info window. The url parameter is a string representing the URL for the link. The optional text parameter determines the text that will appear for the link. If no text is chosen, then the default text is "More information". A complete description of the optional order parameter can be found in the entry for the location.setTxt() function. If no value is chosen for the order parameter, then the default value is 2, which means that the more info link will appear after the title and the descriptive text in the info window. The optional new-window parameter determines if a new window will open when the link is clicked. If new-window is true, then a new window will open when the link is clicked. If a value is not chosen for new-window, then the default value is true. If a value was previously chosen for any of the optional parameters, then that value will be unchanged by this function unless a new value for the parameter is passed in the function call.
location.setInfoLinkTxt(text: string) Return: None
This function sets the text for the more info link in the location's info window.
location.setInfoLinkOrder(order: number) Return: None
This function sets the order of appearance of the more info link in the location's info window.
location.setInfoLinkNew(new-window: boolean) Return: None
This function sets the value of the new-window boolean for the more info link.
location.getInfoLinkUrl() Return: string
This function returns the current URL for the more info link in the location's info window.
location.getInfoLinkTxt() Return: string
This function returns the current text for the more info link in the location's info window.
location.getInfoLinkOrder() Return: number
This function returns the current order of appearance for the more info link in the location's info window.
location.getInfoLinkNew() Return: boolean
This function returns the current value of the new-window boolean for the more info link in the location's info window.
location.delInfoLink() Return: None
This function deletes the current more info link from the location's info window.
