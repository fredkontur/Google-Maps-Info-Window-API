Functions - Map Display

The map functions determine the overall look of the map and whether or not the map gets displayed. All parameters associated with the map have default values, and do not need to be set by the user.

Function Description
setMapCenter(center: location-object) Return: None
This function sets the location of the center of the map. Setting the value of the map center is optional. If no value is chosen, the default value is the first location created by the user.
setMapZoom(zoom: number) Return: None
This function sets the zoom of the map. The higher the zoom value, the more zoomed in the map will be. The zoom value ranges from 0 to about 20 (the maximum value depends on the location being viewed). Setting the value of the zoom is optional. If no value is chosen, the default value is 5.
setLoadMap(loadMap: boolean) Return: None
This function determines if the map will be loaded and displayed. Passing it a value of true means that the map will be loaded and displayed. Setting the value of loadMap is optional. If no value is chosen, the default value is true.
getMapCenter() Return: location-object
This function returns the current location that is the center of the map.
getMapZoom() Return: number
This function returns the current value of the map's zoom.
getLoadMap() Return: boolean
This function returns the current value of the loadMap property.
