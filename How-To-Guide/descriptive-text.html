How-To Guide for Google Maps Info Window API

Functions - Descriptive Text

Below is a list of the functions that can be used to create, delete, or edit the descriptive text in the location's info window. The order property for the descriptive text has not been described previously. The entry for the location.setTxt() function gives a full description of the order property.

Function Description
location.setTxt(text: string, [order: number]) Return: None
This function sets the text field of the location's info window. It is not required for the user to set the descriptive text. If no descriptive text is chosen, then the field will not appear in the info window. The text parameter is a string representing the descriptive text. The optional order parameter determines where the text will appear relative to the other elements in the info window. The lower the order number, the higher the element will appear in the info window. The order parameter is optional for this function. If the user does not choose a value for the descriptive text's order parameter, then, by default, its value is 1, which means that the descriptive text will be the first value to appear after the title in the info window. By default, if there is a title, it always appears at the top of the info window. If the order value was previously set, it will not be affected by this function unless the order parameter is included in the function call.
location.setTxtOrder(order: number) Return: None
This function sets the order of appearance of the descriptive text in the location's info window.
location.getTxt() Return: string
This function returns the current descriptive text for the location's info window.
location.getTxtOrder() Return: number
This function returns the current order of appearance value for the descriptive text.
location.delTxt() Return: None
This function deletes the current descriptive text from the location's info window.
