How-To Guide for Google Maps Info Window API

Functions - Info Window Image

Below is a list of the functions that can be used to create, delete, or edit the image in the location's info window.

Function Description
location.setImg(url: string, [order: number], [width: number]) Return: None
This function sets the image field of the location's info window. It is not required for the user to set the image. If no image is chosen, then the field will not appear in the info window. The url parameter is a string representing the URL for the image. A complete description of the optional order parameter can be found in the entry for the location.setTxt() function. If no value is chosen for the order parameter, then the default value is 3, which means that the image will appear at the bottom of the info window after the title, the descriptive text, and the more info link. The optional width parameter determines the width, in pixels, of the image. If no value is chosen for the image width, then the default value is 300 pixels. If a value was previously chosen for any of the optional parameters, then that value will be unchanged by this function unless a new value for the parameter is passed in the function call.
location.setImgOrder(order: number) Return: None
This function sets the order of appearance of the image in the location's info window.
location.setImgWidth(width: number) Return: None
This function sets the width, in pixels, of the image.
location.getImgUrl() Return: string
This function returns the current URL for the image in the location's info window.
location.getImgOrder() Return: number
This function returns the current order of appearance for the image in the location's info window.
location.getImgWidth() Return: number
This function returns the current value in pixels for the width of the image in the location's info window.
location.delImg() Return: None
This function deletes the current image from the location's info window.
