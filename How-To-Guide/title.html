How-To Guide for Google Maps Info Window API

Functions - Info Window Title

There are three functions related to the info window title - set, get, and delete. These functions are methods of the location object prototype.

Function Description
location.setTitle(title: string) Return: None
This function sets the title field of the location's info window. It is not required for the user to set the title. If no title is chosen, then the field will not appear in the info window. If a title is chosen, then, by default, it will appear at the top of the info window.
location.getTitle() Return: string
This function returns the current value of the location's info window title.
location.delTitle() Return: None
This function deletes the current title from the location's info window.
