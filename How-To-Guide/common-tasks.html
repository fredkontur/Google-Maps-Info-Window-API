How-To Guide for Google Maps Info Window API

Performing Common Tasks

The main thing that needs to be done to make an info window is to create a location on the Google Map for the info window. In order to create a location, use the following command.

var location = addLoc(location-address)

The addLoc() function returns a location object. The location address parameter is a string representing any address that can be recognized by Google Maps. It can be something very specific like a street address or something more general like a city (e.g., “Fresno, CA”) or even a country (e.g., “Peru”).

The Google Maps Info Window API allows you to put four different content fields in the info window.

  1. Title
  2. Descriptive Text
  3. Link for More Information
  4. Image

All of the fields are optional, and you are allowed to put any combination of the four fields in the info window. If you don’t assign any content to the info window for a location that you have created, an empty info window will pop up when the location marker is clicked.

There are two different ways to add content to the info window. If the content fields have been defined when the location is created, you can pass the fields to the addLoc() function.

var location = addLoc(location-address, title, descriptive-text, more-info-url, image-url)

All parameters are passed as strings, and the parameters must be passed to the function in the order listed above. However, you can leave off any number of the parameters at the end of the list. For example, if you don’t wish to pass an image or a link for more information to addLoc(), then you can make the call to the function by leaving out the last two parameters.

Another way to populate the info window content is to use the set methods for the fields. Below is the syntax for the set methods. Note that all of them are methods of the location object prototype and take a string as their parameter.

  1. location.setTitle(title)
  2. location.setTxt(descriptive-text)
  3. location.setInfoLink(more-info-url)
  4. location.setImg(image-url)
