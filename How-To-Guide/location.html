How-To Guide for Google Maps Info Window API

Functions - Location

The 7 functions described in this section are the location creation function, the functions that set and return the location's address, the functions that set and return the location's marker label, and the functions that set and return the value for the maximum width of the location's info window. The functions that are used to create, edit, and delete the contents of the info window are described in later sections.

Function Description
addLoc(location-address: string, [title: string], [descriptive-text: string], [more-info-url: string], [image-url: string]) Return: location-object
This function creates a new location. The only required parameter is the address of the location, entered as a string.
location.setAddress(location-address: string) Return: None
This function sets the address of a location object.
location.setMarkerLabel(label: string) Return: None
This function sets the value of the location's marker label. The marker label is a small text box that appears when the mouse hovers over the location marker. It is not required for the user to set this value. If it is not set, the value defaults to the address given by the user for the location.
location.setMaxWidth(width: number) Return: None
This function sets the value of the maximum width of the location's info window in pixels. It is not required for the user to set this value. If it is not set, the value defaults to 325 pixels.
location.getAddress() Return: string
This function returns the address of a location object.
location.getMarkerLabel() Return: string
This function returns the current marker label for a location object.
location.getMaxWidth() Return: number
This function returns the current value, in pixels, of the maximum width of the location's info window.
