How-To Guide for Google Maps Info Window API

Info Window Style Selectors

All of the elements in the location's info window, including a wrapper containing all of the info window content, are given div classes. Below is a list of the selectors for the different elements in the info window. Please note that the wrapper for the info window content is in another inaccessible container. Because of this, adding a background color or centering elements in the info window may not result in the expected behavior.

Element Class Name Style Selectors
Wrapper for info window content infoBox div, .infoBox
Info window title infoTitle div, h1, .infoTitle
Info window descriptive text infoTxt div, p, .infoTxt
Link to more information infoMore div, p, a, .infoMore
Info window image infoImg div, img, .infoImg
