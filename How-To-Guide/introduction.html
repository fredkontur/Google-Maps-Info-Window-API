How-To Guide for Google Maps Info Window API

Introduction

The Google Maps Info Window API makes it simple for web programmers to add info windows to locations on a Google Map. Info windows are windows that pop up when a user clicks on a location marker on a Google Map. These windows can contain any typical html element including text, images, and links.

Although the Google Maps API provides the capability for creating info windows, it is a rather arduous process to go from a location address, e.g., Toronto, Canada, and associated text, e.g., “Toronto is a beautiful city and is the home of the Maple Leafs and the Blue Jays”, to an info window on a Google Map. This API allows programmers to skip most of those arduous steps. For the Toronto example, the creation of the info window is reduced to 2 straightforward lines of code.

var toronto = addLoc("Toronto, Canada");
toronto.setTxt("Toronto is a beautiful city and is the home of the Maple Leafs and the Blue Jays");

This API could be useful for any number of people and applications – creating an online travel scrapbook, posting the locations and career highlights of former students on an alumni website, displaying an annotated map of famous battlefields for a history of war class, etc.
