How-To Guide for Google Maps Info Window API

Getting Started

You can get started using the Google Maps Info Window API by copying and pasting the following text into an HTML editor.

<!DOCTYPE html>
<html>
  <head>
    <style type="text/css">
        html, body { height: 100%; }
        #map { height: 100%; }
    </style>
  </head>
  <body>
      <div id="map"></div>
      <script src="https://maps.googleapis.com/maps/api/js"></script>
      <script src="https://rawgit.com/fredkontur/Google-Maps-Info-Window-API/master/locations.js"></script>
      <script>
          // Your code here
      </script>
      <script src="https://rawgit.com/fredkontur/Google-Maps-Info-Window-API/master/map.js"></script>
  </body>
</html>

There are several important things to note in the code above.

  1. You need to have a div with id=”map” in order to have a place on the web page for the Google Map.
  2. The styles in the document header define the height of the map div to be 100%. You can change that, but you cannot delete the height definition from the styles because the default height of the div is 0%.
  3. The calls to the JavaScript files need to be in the order listed. Otherwise, the function and variable definitions will occur after they are used in the code and/or the map object will load before the contents of the info windows are passed to it.

If you try to run this HTML file as it is, you’ll get a blank page. That’s because you need to enter code in the spot where it says “Your code here”. In that spot, put the following code.

              var chTitle = "Chicago - A Great Place to Visit!";
              var chDesc = "Make sure to stop by Chicago every chance you get and explore all of the great sites! " +
                  "This city has the best pizza!";
              var moreChInfo = "https://en.wikipedia.org/wiki/Chicago";
              var chImg = "http://www.fodors.com/world/images/destinations/4224/chicago.jpg";
              var chicago = addLoc("Chicago, IL", chTitle, chDesc, moreChInfo, chImg);

After pasting that code, save the HTML file and try opening it in a browser. You should see a web page with a marker on Chicago. If you click on that marker, an info window will open up with a title, some descriptive text, a link, and an image. The map itself should have much of the functionality of a regular Google Map, with zooming and dragging capabilities.

Although we only created one location for this example, you can create as many locations as you like. So, for example, you can add these lines to your code.

              var pikesPeak = addLoc("Pikes Peak, CO"); // Create the location and set its properties afterwards
              pikesPeak.setTitle("Pikes Peak");
              pikesPeak.setImg("http://www.summitleaders.org/wp-content/uploads/2012/01/Pikes-Peak-900x435.jpg");

              var parthenon = addLoc("Parthenon, Greece", "Parthenon"); // Create the location and info window title
              parthenon.setTxt("The Parthenon is an ancient temple in Athens, Greece."); // Now provide the descriptive text

After adding the above code, save the file again and open the page. You'll need to zoom out to see Greece, but all three places – Chicago, Pikes Peak, and the Parthenon – should be marked on the Google Map with info windows that pop up when you click on their markers. You can see an example of this map on the bottom of the page.
